BLOCKDODGE CUSTOM LEVEL GUIDE

Blockdodge Custom Level Guide

Preparations

Your custom level will be a JSON file. So before you begin, create a .txt file.

Level's JSON Structure

            
{
    "route": [
        {
            "pos": [x, y],        // Start position
            "time": x,           // Explosion time
            "blast": x,          // Number of pieces
            "size": x,           // Block size
            "target": [x, y],    // Destination
            "easing": "ease-out",// Keep "ease-out"
            "color": "#ffffff",  // Color hex
            "duration": x        // Wait time for next block
        }
    ]
}

Play in-game

Copy your JSON code and in the menu, paste in-game by clicking to the text box and pressing CTRL + V and pressing enter. Then you can press space to start!
